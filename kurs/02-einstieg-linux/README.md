Bevor wir uns mit Self-Hosting beschäftigen, brauchen wir eine solide Basis: Linux.
Dieses Kapitel führt dich Schritt für Schritt in das Betriebssystem ein, das auf fast jedem
Server der Welt läuft.

## Inhalte

1. [Was ist Linux?](01-was-ist-linux.md) – Geschichte, Aufbau und warum Linux im Server-Bereich dominiert
2. [Distributionen](02-distributionen.md) – Was Distros sind, welche es gibt und welche du wählen solltest
3. [Linux installieren](03-linux-installation.md) – Installation auf echter Hardware, in einer VM oder per WSL

## Lernziele

Nach diesem Kapitel weißt du:

- was Linux ist und wie es sich von Windows unterscheidet
- was eine Distribution ist und welche für Einsteiger geeignet ist
- wie du Linux installierst und die ersten Schritte machst
